# Codes

This folder shows the smallest possible authentication example.

## Example file

- [authentication-basics.js](authentication-basics.js)

## What it does

The file uses hardcoded values and `if` / `else if` / `else` checks to decide whether a login should be accepted or rejected.

## How to run

```bash
node authentication-basics.js
```

## Expected output

```text
Login successful. Access granted.
Login failed. Access denied.
Please enter username and password.
```
