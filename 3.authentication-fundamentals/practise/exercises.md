# Practice Exercises

These exercises stay at the beginner level and only use basic condition checks.

## Beginner

1. Change the stored username from `admin` to your own name.
2. Change the stored password from `1234` to a different value.
3. Add one more example call to `login()` in [codes/authentication-basics.js](../codes/authentication-basics.js).
4. Make the failure message say why the login was rejected.

## Easy thinking questions

1. What happens if the username is correct but the password is wrong?
2. Why is this example not safe for a real website?
3. Why do we check for empty values first?

## Stretch task

Rewrite the `login()` function so it uses one `if`, one `else if`, and one `else` block only.
